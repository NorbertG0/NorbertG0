- Interactive charts and dashboards

- PDF raports creator

- HTML, CSS, Bootstrap

### Weather Discord bot

A Discord bot that provides real-time weather information using external weather API. The bot allows users to check current weather conditions and forecasts directly from Discord channels through simple commands.

**Features:**
- Real-time weather data retrieval
- Weather forecast for selected locations
- Asynchronous processing of user commands
- Integration with external weather APIs
- Location-based weather queries
- Generated weather charts and data visualizations

**Technologies:**
- Python
- Discord.py
